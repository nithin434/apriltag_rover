# AprilTag Rover - Changelog

## [0.1.0] - 2025-12-26

### Added
- Initial project setup with autonomous docking mission demos
- AprilTag follower and mission controller modules
- ROS2 launch configurations for various demo scenarios
- Camera recording and rosbag-to-video conversion utilities
- Synthetic camera simulation for testing
- Comprehensive .gitignore for Python and ROS2 projects

### Features
- Multi-stage autonomous docking sequence
- Real-time AprilTag detection and tracking
- Mission phase management (IDLE, GO_TO_DOCK, LIFT, CHARGER, etc.)
- Simulated visualization of mission progress

### Known Limitations
- Synthetic data generation for initial demos
- Placeholder docking mechanics (framework ready for real sensors)

### Next Steps
- Integration with real hardware sensors
- Enhanced motion planning algorithms
- Improved error handling and recovery mechanisms
